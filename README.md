# Тестовое задание для антивирусной компании

**Проект:** Детектор изменений в apk файлах

---

### Используемый стек технологий:

* **Язык:** Kotlin
* **Архитектура:** MVVM (Model-View-ViewModel)
* **UI-фреймворк:** Jetpack Compose
* **Тестирование:** Unit Tests

---

### Важная информация перед ревью:

Пожалуйста, перед началом проверки задания ознакомьтесь со следующими материалами:

1. **Папка "процесс":** В этой папке находятся дополнительные файлы, описывающие процесс
   тестирования программы.
2. **`readme.pdf`:** В этом PDF-файле содержится подробное описание задания и инструкции по
   использованию.

Спасибо!
